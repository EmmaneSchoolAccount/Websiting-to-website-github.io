Steps in building a image

  • Pick a image (Be careful of the file extension in the url)
  • Record URL, cite it, comment for the date accessed, and record original dimensions
  • Use blockquotes and comment as appropiate
  • Use img-tag with attributes src, alt, title, and style with width-height
